# Шаблон сайта K.B.Net Bootstrap 5

## Описание
Базовый шаблон сайта на **Bootstrap 5** от студии **K.B.Net** (партнерский идентификатор: **kbnet**).

Предназначен для:
- Разработки и тестирования модулей и компонентов 1С-Битрикс
- Быстрого старта новых проектов
- Использования в качестве основы для клиентских сайтов

## Структура шаблона

```
local/templates/kbnet_bootstrap5/
├── header.php                          # Заголовок шаблона
├── footer.php                          # Подвал шаблона
├── empty.php                           # Пустая страница (для AJAX, iframe)
├── description.php                     # Описание шаблона для админки
├── css/
│   ├── template_styles.css             # Основные стили шаблона
│   └── custom.css                      # Пользовательские стили
├── js/
│   └── template_scripts.js             # JavaScript шаблона
├── images/                             # Изображения шаблона
├── fonts/                              # Шрифты
├── components/                         # Переопределения компонентов
│   └── bitrix/
│       ├── breadcrumb/                 # Хлебные крошки
│       ├── menu/                       # Меню (3 варианта)
│       │   ├── horizontal_multilevel/  # Горизонтальное многоуровневое
│       │   ├── left/                   # Левое вертикальное
│       │   └── footer/                 # Подвальное
│       └── search.form/                # Форма поиска
└── lang/ru/                            # Языковые файлы
```

## Компоненты

### Меню
1. **horizontal_multilevel** - Горизонтальное многоуровневое меню с выпадающими списками
2. **left** - Вертикальное левое меню для боковой колонки
3. **footer** - Горизонтальное меню для подвала сайта

### Хлебные крошки (breadcrumb)
Стандартные хлебные крошки Bootstrap 5

### Форма поиска (search.form)
Компактная форма поиска с иконкой

## Установка

1. Скопируйте папку `kbnet_bootstrap5` в директорию `/local/templates/` вашего сайта
2. В административной панели перейдите в: Настройки → Настройки продукта → Сайты → Список сайтов
3. Выберите нужный сайт и укажите шаблон по умолчанию: **K.B.Net Bootstrap 5**

## Использование

### Подключение в коде страницы
```php
<?$APPLICATION->SetTemplate("kbnet_bootstrap5");?>
```

### Отключение хлебных крошек на странице
```php
<?$APPLICATION->SetPageProperty("show_breadcrumb", "N");?>
```

### Отключение левого меню
```php
<?$APPLICATION->SetPageProperty("show_left_menu", "N");?>
```

### Отключение поиска
```php
<?$APPLICATION->SetPageProperty("show_search", "N");?>
```

### Использование пустого шаблона
Для AJAX-ответов, iframe или специальных страниц:
```php
<?$APPLICATION->SetTemplate("kbnet_bootstrap5", "empty.php");?>
```

## Зависимости
- Bootstrap 5.3.2 (подключается через CDN)
- 1С-Битрикс (версия не ниже 19.0.0)

## Кастомизация

### Добавление собственных стилей
Редактируйте файл `/css/custom.css` - он подключается после основных стилей шаблона.

### Изменение структуры
При необходимости скопируйте шаблон в новую папку с уникальным именем и внесите изменения.
